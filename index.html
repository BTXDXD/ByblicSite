О Byblic

Что это?

Byblic — язык для экспериментов и тестов. Интерпретируется для тестов, компилируется в .exe, чтобы напугать друзей.

Как родился?

Написан на Python случайно ночью. Работает только на ПК.

Для кого?

Для тех, кто кодит ради фана и неожиданностей.

Зачем?

Чтобы запутать всех, включая себя.

Секрет успеха

Никто не знает, как он работает, даже Bytespace.

Фичи

Гибрид

Интерпретатор для тестов + компиляция в C.

Поддержка

Язык поддерживается только его сообществом.

Предустановленные штуки

Пользуйся на здоровье.

Поддержка цитрусов

Мандарин — официальный талисман.

Функций нет

Используйте переменные.
